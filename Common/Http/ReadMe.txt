原始版本：
C语言接口，实现撇脚的POST、GET获取数据，要修改库才能设置COOKIE

2019-1-4 更新版本
1.整个接口修改成 CHttp 类，可以方便设置Http Header数据(包括Cookies) 
2.返回数据获取方便，后期扩展方便 
3.加入Range控制下载位置，实现断点续传 
4.另外 HttpDownload 新库调用此库进行多线程下载 

2019-4-14 更新版本
1.修复URL带端口号不能正常访问的问题 
2.HTTPS网站CERT证书过期无法访问的问题，新增SetIgnoreCert接口 
3.添加小网页直接读取整个页面数据而不用设置回调函数，新增接口GetData返回接收数据大小
